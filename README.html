tarchitect

The tarchitect package provides a Shiny widget to help build a targets pipeline. The targets package is documented in The {targets} R package user manual

Installation

You can install the development version of tarchitect from GitHub with:

# install.packages("devtools")
devtools::install_github("PatrickCoyle/tarchitect")
